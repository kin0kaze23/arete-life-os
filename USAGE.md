# AreteLifeOS Usage Guide

## Access
- **URL:** https://arete-life-os.vercel.app
- **iPhone:** Add to Home Screen → Share → Add to Home Screen

## Daily Flow

### Morning (5 min)
1. Unlock vault with passphrase
2. Review dashboard and LifePulseBar (5 colored dots)
3. Check "Do + Watch" section for daily tasks and warnings
4. Log intentions in Log Bar ("Today I will...")

### Throughout Day (1-2 min each)
- Log memories via Log Bar (text, files, voice)
- Check for task updates from AI
- Complete tasks from dashboard
- Log expenses, events, or other updates

### Evening (3-5 min)
1. Review completed tasks
2. Log reflections (mood, energy, gratitude)
3. Optional: Plan tomorrow's priorities

## Quick Commands
- `/sync` - Manually refresh dashboard
- `/ask [question]` - Directly ask Aura AI
- `Capture` - Quick entry point for any memory

## Features by Provider

### Chat & Conversation
- **Powered by:** MiniMax-M2.5 (Optimized for conversational quality)
- **Best for:** Asking questions, getting personalized advice

### Fast Processing
- **Powered by:** Qwen3-coder series (qwen3-coder-next, qwen3-coder-plus)
- **Best for:** Input parsing, quick fact extraction

### Complex Reasoning
- **Powered by:** Qwen3-max-2026-01-23
- **Best for:** Deep task generation, complex insight analysis

### Long-form Synthesis
- **Powered by:** Kimi-k2.5
- **Best for:** Strategic briefings, life snapshots

## Tips
- Be specific in logs: "Spent $50 at Starbucks for business meeting" vs "Lunch"
- Review AI extractions early to improve accuracy over time
- Check LifePulseBar weekly for dimension trends
- Use dimension tags on tasks to track which life area you're improving
