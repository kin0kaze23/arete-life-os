# Product North Star

## Purpose

A Life OS for daily family use (me/spouse/children) that:

- tells us what to do today (what/why/how)
- tells us what to watch out for (why/impact/next prevention)
- stays calm, clear, and hyper-personalized

## Primary user questions after onboarding

1. What should I be doing, why, and how?
2. What should I look out for, why, and what happens if I ignore it?

## Non-negotiable decisions

### 1) Log Bar = universal command center

- Capture/upload: text, files, images, links
- CRUD: Memory Vault, Profile/Identity fields, Core Loop config
- Execute: plan my day, summarize/extract, attach/link, export/import
- States: draft / processing / success / needs-review / failed
- Reactive updates across the app (no manual refresh)
- "Capture" is not top-level; it's Log Bar default behavior
- "Portability" is not top-level; it's export/import actions from Log Bar or log items

### 2) Dashboard = Do + Watch across horizons

- Always answer: Do (tasks) + Watch (risks)
- Horizon layers: Now (today) / Soon (7–14 days) / Always (ongoing)

### 3) Recommendations must be executable

Every recommendation must include:

- what / why / steps / time & effort / when / inputs / definition-of-done / risks / follow-up
  Grounded in stored data. Uncertainty goes to Needs Review.

## MVP definition

MVP is "stable daily loop":

- Onboarding + Profile config
- Dashboard (Do/Watch + horizon switcher)
- Log Bar ingestion + Needs Review
- Domain recommendations + Plan My Day time-blocking
- Nightly review feedback loop
- Future events -> milestone prep tasks + watch-outs
