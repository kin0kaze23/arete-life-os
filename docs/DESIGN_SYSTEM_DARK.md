# Design System: Premium Dark (MVP)

## Principles

- premium, calm, minimal
- high readability, no harsh neon
- subtle depth and purposeful motion

## Tokens (conceptual)

- bg: near-black
- surface: dark slate
- surface-elevated: slightly lighter
- text-primary: near-white
- text-secondary: muted gray
- border: low-contrast
- accent: one tasteful highlight color
- status: success/warn/danger/info (muted, accessible)

## Components rules

- Cards: soft border, subtle shadow, consistent padding
- Chips: compact, clear states (default/selected/disabled)
- Buttons: primary/secondary/ghost only
- Lists: strong hierarchy, visible click affordance
- Drawers/Modals: used for details, not navigation

## Accessibility basics

- visible focus ring
- sufficient contrast for text + badges
- tap targets >= 44px
