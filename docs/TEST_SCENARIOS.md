# Must-pass Scenarios (Pre-Deployment)

## Core daily loop

1. Onboarding completes; profile fields saved and editable
2. Dashboard shows Do + Watch; Plan My Day generates time-block plan
3. Completing a task updates dashboard immediately

## Future events (anticipatory guidance)

4. Add event: "Korea trip March 2026" -> milestones generated (T-60/T-30/T-14/T-7)
5. As time approaches, tasks move from Soon -> Now automatically
6. Watch-outs appear if prep tasks overdue

## Memory vault ingestion

7. Upload PDF lab report -> classified to Health + person; shows as source in recommendations
8. Uncertain classification -> Needs Review asks minimal questions and routes correctly
9. Duplicate upload -> dedup/versioning behavior works

## Quality & resilience

10. Empty states look intentional and guide next action
11. Loading/error states don’t break the flow
12. Large lists don’t lag noticeably; pagination or virtualization exists if needed
