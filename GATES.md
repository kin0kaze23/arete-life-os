# Verification Gates — AreteLifeOS

```bash
# Gate 1: Format
npm run format:check

# Gate 2: Lint
npm run lint

# Gate 3: Typecheck
npm run typecheck

# Gate 4: Unit/Integration Tests
npm run test

# Gate 5: Build
npm run build
```

## Release expectation

- `npm run doctor` must pass before PR handoff or release.
